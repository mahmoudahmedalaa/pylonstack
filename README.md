# Pylon - Tech Stack Engine

Pylon is an intelligent Tech Stack Engine that understands your project requirements and instantly generates the perfect architecture, complete with interactive blueprints.

## Features
- **AI-Powered Decisions**: Recommend tools based on your project constraints.
- **Interactive Blueprints**: Visualize your entire stack structurally. Drag, drop, and customize layers.
- **Enterprise Validated**: Recommendations backed by real-world usage and compatibility guarantees.
- **Curated Catalog**: A comprehensive database of modern development tools.

## Tech Stack
- Frontend: Next.js 15 (App Router, Turbopack), React 19, Tailwind CSS
- UI Components: Framer Motion, Radix UI, Lucide Icons
- State Management: Zustand
- Database: Supabase (PostgreSQL), Drizzle ORM
- AI Integration: Vercel AI SDK, Google Gemini

## Local Development
1. Clone the repository
2. Run `npm install`
3. Set up your `.env.local` with Supabase and Gemini keys
4. Run `npm run dev` to start the development server
